# 多进程/多实例：使用 HappyPack 解析资源

原理：每次 webpack 解析一个模块，HappyPack 会将它及它的依赖分配给 worker 线程中

# 代码实例

exports.plugins = [
    new HappyPack({
        id: 'jsx',
        threads: 4,
        loaders: ['babel-loader']
    }),
    new HappyPack({
        id: 'styles',
        threads: 2,
        loaders: ['style-loader', 'css-loader', 'less-loader']
    })
]
